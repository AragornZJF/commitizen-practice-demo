# comitizen-practice-demo
Vue 项目 commitizen + husky, git commit 提交信息规范校验 demo

## 初始化项目
```bash
npm install @vue/cli -g # 安装 @vue/cli
vue create comitizen-practice-demo
# [Vue 3] typescript, router, vuex, eslint, unit-mocha) 
```
## Project setup
```
npm install
```
